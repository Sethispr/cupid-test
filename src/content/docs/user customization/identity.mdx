View your identity card with `=identity` or the alias `=id`. Customize it using the `=select` command with the filters.

## Customization Options

<Tabs>
<TabItem label="Avatar">

```sh
=select -a [avatar name]
```

**Example**: `=select -a hanako`

Changes the avatar image on your identity card to Hanako.

</TabItem>
<TabItem label="Border">

```sh
=select -b [border name]
```

**Example**: =select -b hunter

Sets the Hunter Season border style to your card.

</TabItem>
<TabItem label="Background">

```sh
=select -bg [background name]
```

**Example**: =select -bg bond

Sets the background Bond to your card.

</TabItem>
<TabItem label="Font">

```sh
=select -f [font name]
```

**Example**: =select -f automata

Changes the text font to Automata style.

</TabItem>
<TabItem label="Color">

```sh
=select -c [hex color code, e.g. #fff271]
```

**Example**: =select -c #55d1d0

Sets the identity and profile message embed color to #55d1d0.

</TabItem>
<TabItem label="Badge">

```sh
=select -badge [badge name]
```

**Example**: =select -badge christmas2024

Sets the Christmas2024 badge to your card and profile. You can only select 1 badge.

</TabItem>
</Tabs>

Check your owned cosmetics with `=vault` or the alias `=vlt`.

You can also edit your identity directly via the `=vlt` output embed buttons. Select an option and enter the cosmetic name (e.g. Nene Yashiro).

### Examples

**Do**:

- `=select -a nene` (equips Nene Yashiro avatar)
- `=select -c #55d1d0` (uses #55d1d0 for your embed color)
- `=vlt` (check owned cosmetics)

**Don't**:

- `=select -bg` (missing background name)
- Using invalid hex codes (e.g. `#gggggg`).
- Trying to use unowned/invalid cosmetics.
